# Online Furniture Store

> A full-stack online furniture marketplace built using the MERN stack (MongoDB, Express.js, React, Node.js). 
> The platform allows users to browse furniture products, place orders, and enables administrators to manage inventory, product images, and overall store operations through a dedicated admin dashboard.
